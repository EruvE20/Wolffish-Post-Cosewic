# Wolffish-Post-Cosewic
Shared code for wolffish assessment project

This repo contains the code for generating tables and figures for the DFO Post-COSEWIC assessment of Atlantic Wolffish on the Scotian Shelf. Users are assumed to have access to the following databases through Oracle:
- ISDB catches database
- RV (groundfish) database
- Marfis database

If you do not have access, please contact your administrator. Data can be queried through SQL or the [Maritimes Data Wrangling package](https://github.com/Maritimes/Mar.datawrangling/wiki).
